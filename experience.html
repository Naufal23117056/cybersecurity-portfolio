Computer Networking

I configured networks in simulations via GNS3 as well as physically. Next, I was able to set up a network using a layer 3 switch and connected three routers together using EIGRP. With this network set up, I was able to simulate a MAC address spoofing attack on a router, feeding it false mac addresses that do not exist.
